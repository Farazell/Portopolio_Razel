## 4. Kebutuhan Non-Fungsional (Non-Functional Requirements)

### 4.1. Tema & Panduan Gaya UI/UX (Gelap Neon Kreatif)
* **Warna Latar Belakang:** Hitam Pekat (*Deep Charcoal/Pure Black* `#0B0C10` atau `#121212`) untuk memberikan kontras maksimal pada elemen kreatif Anda.
* **Warna Aksen (Neon Glow):**
    * **Hijau Neon / Cyan Siber (`#00F5D4` atau `#39FF14`):** Digunakan pada *hover effect* tombol kontak, teks penting, atau *border* kartu proyek untuk memberikan kesan canggih, aktif, dan berteknologi tinggi.
    * **Aksen Muted Green/Emerald:** Mengadopsi corak warna khas dari dokumen portofolio cetak Anda untuk mempertahankan konsistensi *personal branding*.
* **Tipografi:** Font Sans-Serif tegas seperti *Space Grotesk*, *Syne*, atau *Poppins* dengan variasi ketebalan (*bold font tracking*) untuk judul utama.

### 4.2. Performa Media & Kode
* [cite_start]**Sematkan Video Integrasi:** Untuk bagian sinematografi, gunakan pemutar video kustom yang ringan atau integrasi YouTube/Vimeo API agar tidak memperlambat *first contentful paint* (FCP) website[cite: 375].
* **Framework Rekomendasi:** **Vue.js / Nuxt.js** (karena Anda familier dengan ekosistem Vue dan JavaScript) dikombinasikan dengan **Tailwind CSS** untuk mempermudah pembuatan efek bayangan neon (*box-shadow glow*).
