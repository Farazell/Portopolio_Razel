
```bash

```

---

## 🚀 Menjalankan Project

### Menggunakan Antigravity

1. Buka aplikasi Antigravity.
2. Pilih **Open Folder**.
3. Buka folder project:

```text
Portopolio_Razel
```

4. Pilih file:

```text
index.html
```

5. Jalankan menggunakan fitur **Preview** atau **Live Preview** yang tersedia di Antigravity.

---

### Menjalankan Langsung di Browser

#### Windows

```powershell
start index.html
```

#### Linux

```bash
xdg-open index.html
```

#### macOS

```bash
open index.html
```

## 📤 Upload Perubahan ke GitHub

Cek status repository:

```bash
git status
```

Tambahkan semua perubahan:

```bash
git add .
```

Commit perubahan:

```bash
git commit -m "Update portfolio"
```

Push ke GitHub:

```bash
git push origin main
```

---

## 🌿 Branch Management

Membuat branch baru:

```bash
git checkout -b fitur-baru
```

Pindah ke branch lain:

```bash
git checkout main
```

Menggabungkan branch:

```bash
git merge fitur-baru
```

Menghapus branch:

```bash
git branch -d fitur-baru
```
